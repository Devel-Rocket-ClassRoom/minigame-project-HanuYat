# 주요 에셋 및 컨텍스트
- **대상 개구부 1 (앞문)**: `x=6.25`, `z` 범위 `13.75` ~ `16.25`. 너비 2.5m, 높이 2.5m.
- **대상 개구부 2 (뒷문)**: `x=6.25`, `z` 범위 `31.25` ~ `33.75`. 너비 2.5m, 높이 2.5m.
- **새로운 문 프리팹**: `Assets/Imported/Cute_Classroom_Set/Prefabs/Door.prefab` (크기: 0.89m x 2.06m x 0.06m).
- **기존 문**: `Classroom_01_FrontDoor`, `Classroom_01_BackDoor` (부모 오브젝트: `Door`).

# 구현 단계
1. **개구부 준비**:
    - 기존의 `Classroom_01_FrontDoor` 및 `Classroom_01_BackDoor` 오브젝트를 비활성화하거나 삭제합니다.
2. **앞문 미닫이 문 설치**:
    - `Door.prefab`을 두 번 생성합니다.
    - 이름을 `FrontDoor_Left`, `FrontDoor_Right`로 변경합니다.
    - **스케일(Scale)**:
        - 너비(X): `1.405` (0.89m -> 1.25m로 확장하여 개구부 절반을 채움).
        - 높이(Y): `1.214` (2.06m -> 2.5m로 확장하여 벽 높이에 맞춤).
        - 두께(Z): `1.0`.
    - **회전(Rotation)**: Y축으로 `90`도 회전하여 문 너비가 씬의 Z축과 정렬되도록 합니다.
    - **배치(Positioning)**:
        - `FrontDoor_Left`: `(6.22, 0, 14.375)` (앞문 개구부 왼쪽 절반 중앙, 미닫이 레일을 고려해 X축 약간 오프셋).
        - `FrontDoor_Right`: `(6.28, 0, 15.625)` (앞문 개구부 오른쪽 절반 중앙, 미닫이 레일을 고려해 X축 약간 오프셋).
3. **뒷문 미닫이 문 설치**:
    - `Door.prefab`을 두 번 생성합니다.
    - 이름을 `BackDoor_Left`, `BackDoor_Right`로 변경합니다.
    - **스케일**: 앞문과 동일 (`1.405, 1.214, 1.0`).
    - **회전**: Y축 `90`도.
    - **배치**:
        - `BackDoor_Left`: `(6.22, 0, 31.875)` (뒷문 개구부 왼쪽 절반 중앙).
        - `BackDoor_Right`: `(6.28, 0, 33.125)` (뒷문 개구부 오른쪽 절반 중앙).
4. **그룹화**:
    - 모든 새로운 문 오브젝트를 기존 `Classroom_01/Door` 부모 오브젝트 아래로 자식화하여 계층 구조를 유지합니다.

# 검증 및 테스트
- **시각적 확인**: 문이 벽 사이의 2.5m 공간을 빈틈없이 채우는지, 벽과 자연스럽게 연결되는지 확인합니다.
- **정렬 확인**: 문의 바닥이 바닥면(`y=0`)에 닿아 있고, 상단이 천장/벽 높이(`y=2.5`)와 일치하는지 확인합니다.
- **슬라이딩 확인**: 에디터에서 문을 움직여 보며 두 문이 서로 겹치지 않고 (X축 오프셋을 통해) 자연스럽게 교차할 수 있는지 확인합니다.
