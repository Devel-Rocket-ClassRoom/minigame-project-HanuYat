# 주요 에셋 및 컨텍스트
- 프리팹: `Assets/Imported/LowPolyInterior/Prefabs/Light/Light_01.prefab`
- 교실 천장: `Classroom/Ceiling` (범위: 10x20 유닛, 중심점 11.25, 2.49, 23.75)
- 활성 씬: `Assets/Scenes/Main Scene.unity`

# 구현 단계
1. **조명 부모 오브젝트 준비**: `Classroom` 오브젝트 하위에 `Lights`라는 이름의 새로운 GameObject를 생성하여 조명들을 체계적으로 관리합니다.
2. **그리드 계산**:
    - 천장 영역: X [6.25, 16.25], Z [13.75, 33.75].
    - 그리드 크기: 가로 8 x 세로 16 = 총 128개 조명.
    - 간격: X, Z 양방향 모두 1.25 유닛.
    - 높이 (Y): 2.52 (조명 프리팹의 피벗이 상단에 있고 높이가 0.1이므로, 천장 하단인 Y=2.42에 맞추기 위해 설정).
3. **조명 배치**: 반복문을 통해 계산된 위치에 `Light_01` 프리팹을 배치합니다.
    - X 위치: `6.25 + (i + 0.5) * 1.25`
    - Z 위치: `13.75 + (j + 0.5) * 1.25`
    - Y 위치: `2.52`
4. **마무리**: 모든 조명 인스턴스가 `Classroom/Lights`의 자식으로 설정되었는지 확인합니다.

# 검증 및 테스트
- 수동 확인: 씬 뷰에서 조명이 그리드에 맞춰 잘 배치되었는지, 천장에 매립된 느낌(flush)으로 설치되었는지 확인합니다.
- 계층 구조 확인: `Classroom/Lights` 하위에 128개의 `Light_01` 인스턴스가 생성되었는지 확인합니다.
