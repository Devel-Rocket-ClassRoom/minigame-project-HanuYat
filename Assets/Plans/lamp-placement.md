# 주요 에셋 및 컨텍스트
- **조명 프리팹**: `Assets/Imported/LowPolyInterior/Prefabs/Light/Lamp_07.prefab`
- **대상 위치**: 메인 복도의 좌측 벽면 (X ≈ -1.25), 창문 그룹 사이 공간.
- **창문 위치 식별**: Z축 좌표 2.5, 10.0, 17.5, 25.0, 32.5, 40.0, 47.5 지점에 위치한 `WallFloor1_09` 오브젝트들.

# 구현 단계
1. **배치 지점 확인**:
   - 좌측 창문(`WallFloor1_09`)들은 Z축 기준 2.5, 10.0, 17.5, 25.0, 32.5, 40.0, 47.5 지점에 배치되어 있습니다.
   - 각 창문 사이의 중간 지점에 조명을 배치합니다:
     - Z = 6.25
     - Z = 13.75
     - Z = 21.25
     - Z = 28.75
     - Z = 36.25
     - Z = 43.75
2. **조명(Lamp) 인스턴스 생성**:
   - `Main Corridor` 하위에 `Left_Wall_Lamps`라는 부모 오브젝트를 생성합니다.
   - 6개의 `Lamp_07` 프리팹 인스턴스를 다음 좌표에 생성합니다:
     - 위치: `(-1.15, 1.7, Z)` (위에서 계산된 Z값들)
     - 회전: `(0, 90, 0)` (벽면 방향에 맞춰 정렬)
3. **Light 컴포넌트 설정**:
   - 각 생성된 조명 인스턴스에 `Light` 컴포넌트를 추가합니다.
   - 속성 설정:
     - **Type (유형)**: Point (포인트)
     - **Range (범위)**: 4.0
     - **Intensity (강도)**: 1.5
     - **Color (색상)**: 따뜻한 흰색 (예: Hex #FFF4CC)
     - **Mode (모드)**: Baked (간접 조명 느낌을 위해 권장) 또는 Realtime.
4. **최종 조정**:
   - 조명이 벽면 메쉬에 겹치거나 묻히지 않는지 확인합니다.

# 검증 및 테스트
- 씬 뷰에서 조명이 창문 사이 중앙에 올바르게 위치했는지 확인합니다.
- 각 인스턴스에 `Light` 컴포넌트가 추가되고 설정값이 올바르게 입력되었는지 확인합니다.
- (수동 확인) 조명을 켰을 때 복도의 시각적 분위기가 의도한 대로 표현되는지 확인합니다.
