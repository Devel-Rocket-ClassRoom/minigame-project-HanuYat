# 프로젝트 개요
- 게임 제목: 한국식 교실 - 밤의 학교 (Night School)
- 핵심 컨셉: 교실 창밖 풍경을 밤으로 설정하고 으스스한(Spooky) 분위기를 연출함.
- 렌더 파이프라인: URP (PC_RPAsset).

# 게임 메커니즘
## 환경 설정 (Environment Setup)
- 기본 조명을 낮추고 달빛 느낌의 조명을 추가하여 밤 분위기 조성.
- 안개(Fog)를 추가하여 시야를 제한하고 으스스한 느낌을 극대화.

# UI
- 해당 없음.

# 주요 에셋 및 컨텍스트
- **조명**: `Directional Light` (달빛 역할).
- **포스트 프로세싱**: `Global Volume` (Bloom, Vignette, Color Adjustments).
- **생성할 에셋**: 으스스한 마른 나무 (Spooky Dead Tree), 낡은 울타리 (Old Creepy Fence).
- **위치**: 교실 외부 공간 (창문 및 문 밖).

# 구현 단계
1. **기본 환경 조정**:
    - `Directional Light`의 강도를 낮추고(0.2 - 0.3) 색상을 차가운 파란색/은색 계열로 변경합니다.
    - `RenderSettings.fog`를 활성화하고, 안개 색상을 어두운 남색이나 검은색으로 설정합니다.
    - `RenderSettings.ambientLight`를 어둡게 조정합니다.
2. **으스스한 에셋 생성 및 배치**:
    - `GenerateAsset` 도구를 사용하여 'Spooky Dead Tree'와 'Old Creepy Fence'를 생성합니다.
    - 생성된 에셋을 교실 창문 너머(창문 밖 공간)에 배치하여 배경을 채웁니다.
3. **포스트 프로세싱 강화**:
    - `Global Volume`의 프로파일에 다음 효과를 추가/수정합니다:
        - **Vignette**: 가장자리를 어둡게 하여 긴장감 조성.
        - **Bloom**: 은은한 빛 번짐 효과.
        - **Color Adjustments**: 채도를 낮추고 전체적인 톤을 차갑게 조정.
4. **창문 밖 조명 (옵션)**:
    - 창밖 특정 위치에 어두운 보라색이나 녹색의 약한 `Point Light`를 배치하여 기괴한 분위기를 추가할 수 있습니다.

# 검증 및 테스트
- **분위기 확인**: 씬 뷰와 게임 뷰에서 전체적인 밝기와 색감이 "으스스한 밤" 느낌을 주는지 확인합니다.
- **가시성 체크**: 안개가 너무 짙어 아무것도 보이지 않거나, 너무 밝아 밤 느낌이 나지 않는지 조정합니다.
- **에셋 배치**: 창문을 통해 보았을 때 나무와 울타리가 적절한 구도로 배치되었는지 확인합니다.
